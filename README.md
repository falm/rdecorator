# RDecorator

Rdecorator is an Pyhton like decorator use for AOP pattern programming

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rdecorator', github: 'https://github.com/falm/rdecorator.git'
```

And then execute:

    $ bundle
